# 개인정보 처리방침 초안

약국쇼핑몰 구매도우미는 사용자의 구매 계산과 장바구니 담기 보조를 위해 동작합니다.

## 수집하지 않는 정보

- 쇼핑몰 로그인 아이디
- 비밀번호
- 결제정보
- 카드정보
- 주민등록번호 등 고유식별정보
- 서버로 전송되는 구매내역

## 브라우저 안에서 처리되는 정보

다음 정보는 사용자의 브라우저 안에서만 처리됩니다.

- 상품명
- 업체명
- 단가
- 수량
- 최저주문금액
- 쿠폰 조건
- 추천 구매 조합

## 외부 전송

현재 버전은 별도 서버로 데이터를 전송하지 않습니다.

## 권한 사용 목적

확장 프로그램은 HMP몰 화면에서 검색창, 상품, 업체 행, 장바구니 버튼을 찾기 위해 HMP몰 페이지 접근 권한을 사용합니다.

## 결제

본 도구는 장바구니 담기까지만 보조하며 결제를 자동으로 진행하지 않습니다. 사용자는 최종 결제 전에 장바구니 내용을 직접 확인해야 합니다.
