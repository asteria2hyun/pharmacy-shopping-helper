# 약국쇼핑몰 구매도우미 확장 프로그램

계산기에서 만든 `hmp-auto-cart-v1` 자동담기 데이터를 HMP몰 탭으로 보내 장바구니 담기를 시도하는 확장 프로그램입니다.

## 설치

1. Chrome 주소창에 `chrome://extensions` 입력
2. 오른쪽 위 `개발자 모드` 켜기
3. `압축해제된 확장 프로그램을 로드` 클릭
4. 이 폴더 선택:
   `C:\Users\joosung\Documents\Codex\2026-05-08\hmp\hmp-extension`

## 사용

1. `hmp-helper.html`에서 구매 조합을 계산합니다.
2. `자동담기 데이터 복사`를 누릅니다.
3. HMP몰에 로그인합니다.
4. Chrome 확장 프로그램에서 `약국쇼핑몰 구매도우미`를 엽니다.
5. `클립보드 붙여넣기`를 누릅니다.
6. `자동 담기 시작`을 누릅니다.

## 현재 한계

- HMP몰의 검색창, 상품명, 업체명, 장바구니 버튼을 화면 텍스트 기반으로 찾습니다.
- HMP몰 화면 구조가 다르면 실패할 수 있습니다.
- 결제는 자동으로 하지 않습니다. 장바구니 담기까지만 시도합니다.
- 실패 항목은 팝업 상태 메시지로 확인하고, 다음 버전에서 HMP 화면 구조에 맞게 선택자를 보강합니다.

## 공개 배포 전 확인

- `manifest.json`의 `version` 업데이트
- 개인정보 처리방침 URL 준비
- 실제 HMP 계정에서 소량 테스트
- 결제 자동화가 없음을 사용자에게 명확히 안내
