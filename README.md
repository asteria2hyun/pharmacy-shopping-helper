# 약국쇼핑몰 구매도우미

약국 쇼핑몰에서 여러 상품을 구매할 때, 업체별 가격과 최저주문금액, 쿠폰 조건을 반영해 구매 조합을 계산하는 도구입니다.

현재는 HMP몰을 먼저 지원합니다.

## 구성

- `hmp-helper.html`  
  가격 비교와 구매 조합 계산을 하는 웹 계산기입니다.

- `hmp-extension/`  
  HMP몰 탭에서 계산 결과를 장바구니 담기 작업으로 연결하는 Chrome/Whale 확장 프로그램입니다.

## 추천 배포 방식

1. GitHub 저장소에 이 프로젝트를 올립니다.
2. GitHub Pages로 `hmp-helper.html` 계산기를 공개합니다.
3. 확장 프로그램은 초기에는 `압축해제된 확장 프로그램` 설치 방식으로 테스트합니다.
4. 안정화 후 Chrome Web Store 또는 Whale 확장앱 배포를 검토합니다.

## 사용자 흐름

1. 계산기에서 HMP 상품과 업체별 가격을 입력하거나 가져옵니다.
2. 수량, 최저주문금액, 쿠폰을 확인합니다.
3. 추천 구매 조합을 확인합니다.
4. `자동담기 데이터 복사`를 누릅니다.
5. HMP몰에 로그인한 뒤 확장 프로그램에서 `자동 담기 시작`을 누릅니다.
6. 장바구니에 담긴 내용을 직접 확인하고 결제합니다.

자세한 사용 방법은 [USER_GUIDE.md](./USER_GUIDE.md)를 참고하세요.

## 중요한 원칙

- 결제는 자동으로 진행하지 않습니다.
- 로그인 아이디, 비밀번호, 결제정보를 저장하지 않습니다.
- 계산기 데이터는 사용자의 브라우저 안에서 처리합니다.
- HMP몰 화면 구조가 바뀌면 자동담기 기능이 실패할 수 있습니다.
- 각 쇼핑몰의 이용약관과 정책을 확인한 뒤 사용해야 합니다.

## 개발/테스트

로컬에서 `hmp-helper.html`을 브라우저로 열면 바로 계산기를 테스트할 수 있습니다.

확장 프로그램 설치:

1. Chrome: `chrome://extensions`
2. Whale: `whale://extensions`
3. 개발자 모드 켜기
4. `압축해제된 확장 프로그램/확장앱 로드`
5. `hmp-extension` 폴더 선택

## 배포 전 체크리스트

- 실제 HMP 상품 1개로 자동담기 테스트
- 검색창, 상품 선택, 업체 행, 장바구니 버튼 인식 확인
- 실패 시 사용자에게 명확한 메시지 표시
- 개인정보 처리방침 준비
- 사용 설명서와 면책 문구 정리
- 확장 프로그램 권한 최소화 검토
