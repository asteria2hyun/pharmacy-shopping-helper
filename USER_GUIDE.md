# 약국쇼핑몰 구매도우미 사용 설명서

이 도구는 HMP몰에서 여러 상품을 구매할 때 업체별 가격, 최저주문금액, 쿠폰 조건을 비교해 구매 조합을 추천하고, 확장 프로그램으로 장바구니 담기를 보조합니다.

## 전체 흐름

1. 계산기에서 구매할 상품을 입력합니다.
2. 업체별 가격, 수량, 최저주문금액, 쿠폰을 확인합니다.
3. 추천 구매 조합을 확인합니다.
4. 자동담기 데이터 복사를 누릅니다.
5. HMP몰에 로그인합니다.
6. 확장 프로그램에서 자동 담기 시작을 누릅니다.
7. 장바구니 내용을 직접 확인하고 결제합니다.

결제는 자동으로 진행되지 않습니다.

## 1. 계산기 열기

계산기 주소:

`https://asteria2hyun.github.io/pharmacy-shopping-helper/hmp-helper.html`

계산기에서 할 일:

- 상품 추가
- 검색어 입력
- 선택 상품명 입력
- 업체별 단가 입력
- 업체별 최저주문금액 입력
- 수량 조절
- 쿠폰이 있으면 쿠폰 입력

## 2. 상품 입력 방법

상품마다 다음 정보를 확인합니다.

- 검색어: HMP에서 검색할 단어
- 선택 상품명: HMP에 보이는 실제 상품명
- 수량: 구매하려는 수량
- 업체명: HMP 가격표의 업체명
- 단가: 업체별 단가
- 최저주문금액: 업체별 주문 가능 최소 금액
- 재고: 품절 여부 확인용

선택 상품명은 자동담기에서 중요합니다. HMP 검색 결과에 보이는 상품명과 최대한 비슷하게 입력해야 합니다.

## 3. 업체별 합계 확인

오른쪽의 업체별 합계에서 업체별 주문 가능 여부를 확인합니다.

- 주문 가능: 최저주문금액을 넘은 업체
- 부족: 최저주문금액에 못 미친 업체

업체명 앞의 체크박스를 선택하면, 체크한 업체만 자동담기 데이터로 복사됩니다.

예를 들어 헬스인팜만 체크하면 헬스인팜 구매 목록만 자동담기 데이터에 들어갑니다.

## 4. 대안 조합 확인

검토할 대안에는 다른 구매 조합이 순위별로 표시됩니다.

대안 카드를 클릭하면:

- 구매 목록이 해당 조합으로 바뀝니다.
- 업체별 합계도 해당 조합으로 바뀝니다.
- 자동담기 데이터도 선택한 조합 기준으로 만들어집니다.

공동 1순위가 여러 개일 수 있습니다. 원하는 조합을 클릭해서 선택한 뒤 자동담기 데이터를 복사하면 됩니다.

## 5. 자동담기 데이터 복사

오른쪽 아래 HMP 자동담기 영역에서 `자동담기 데이터 복사`를 누릅니다.

복사 전 확인:

- 업체별 합계에서 담을 업체가 체크되어 있는지
- 선택 상품명이 실제 HMP 상품명인지
- 수량이 맞는지
- 최저주문금액 조건이 맞는지

## 6. 확장 프로그램 설치

Chrome 또는 Whale에서 설치할 수 있습니다.

Chrome:

1. `chrome://extensions` 접속
2. 개발자 모드 켜기
3. 압축해제된 확장 프로그램 로드
4. `hmp-extension` 폴더 선택

Whale:

1. `whale://extensions` 접속
2. 개발자 모드 켜기
3. 압축해제된 확장앱 로드
4. `hmp-extension` 폴더 선택

설치 후 확장 이름은 `약국쇼핑몰 구매도우미`입니다.

## 7. HMP 자동담기 실행

1. HMP몰에 직접 로그인합니다.
2. HMP몰 탭을 열어둡니다.
3. 브라우저 오른쪽 위 확장 프로그램 아이콘을 누릅니다.
4. `약국쇼핑몰 구매도우미`를 엽니다.
5. `클립보드 붙여넣기`를 누릅니다.
6. 담기 예정 업체 수와 품목 수를 확인합니다.
7. `자동 담기 시작`을 누릅니다.
8. HMP 화면 오른쪽 아래 진행 패널을 확인합니다.

자동담기는 검색, 상품 선택, 업체 행 찾기, 수량 입력, 장바구니 담기 순서로 진행됩니다.

## 8. 반드시 확인할 것

자동담기 후에는 HMP 장바구니에서 직접 확인해야 합니다.

- 상품이 맞는지
- 업체가 맞는지
- 수량이 맞는지
- 쿠폰 적용 조건이 맞는지
- 최저주문금액을 만족하는지
- 품절이나 주문마감이 없는지

결제는 직접 확인 후 진행합니다.

## 9. 자주 생기는 문제

### 자동담기 데이터 복사가 안 됨

원인:

- 선택 상품명이 업체 가격행처럼 보이는 경우
- 예: `대지인팜 2,300 6,971`

해결:

- 상품 카드의 선택 상품명을 HMP 실제 상품명으로 수정합니다.

### 확장에서 HMP 탭이 아니라고 나옴

해결:

- HMP몰 탭을 클릭한 상태에서 확장 프로그램을 다시 엽니다.

### HMP 진행 패널이 안 뜸

해결:

1. 확장 관리 페이지에서 확장 새로고침
2. HMP몰 탭 새로고침
3. 다시 자동 담기 시작

### 상품을 못 찾음

해결:

- 검색어를 HMP에서 잘 검색되는 단어로 바꿉니다.
- 선택 상품명을 HMP 검색 결과의 상품명과 더 비슷하게 수정합니다.

### 업체 행을 못 찾음

해결:

- 계산기 업체명이 HMP 가격표의 업체명과 같은지 확인합니다.
- 띄어쓰기나 특수문자를 줄여서 다시 시도합니다.

## 10. 주의사항

- 이 도구는 장바구니 담기 보조 도구입니다.
- 결제를 자동으로 하지 않습니다.
- 로그인 아이디와 비밀번호를 저장하지 않습니다.
- 쇼핑몰 화면 구조가 바뀌면 자동담기가 실패할 수 있습니다.
- 최종 주문 전에는 반드시 사람이 확인해야 합니다.
